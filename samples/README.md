# Samples — 合成資料分析範例

此目錄保存用合成資料（`data/orders.csv`）跑完的分析結果，供對照參考。

## 內容

- `notebooks/` — 舊版 layer-based notebooks（已含執行結果）
- `reports/` — 圖表 PNG + review summary
- `create_notebooks.py` — 舊版 notebook 生成腳本（已不使用）

## 用途

- 看「跑完大概長什麼樣」— 對照你在真實資料上的結果
- 理解每個分析步驟的邏輯和產出格式
- **不要直接用這些 notebook 跑真實資料** — 請用 `notebooks/step*.ipynb`
